🚀 快速启动
────────────────────────────────────────────────────────

方式1: 一键安装
  chmod +x setup.sh
  ./setup.sh

方式2: 手动启动

  # 后端（终端1）
  cd backend
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  python run.py

  # 前端（终端2）
  cd frontend
  npm install
  npm run dev

访问: http://localhost:5173

⚠️ 注意事项
────────────────────────────────────────────────────────
1. 首次运行会下载PaddleOCR模型（~100MB）
2. OCR识别受图片质量影响
3. 需要联网获取实时估值数据
4. 数据存储在本地SQLite数据库
